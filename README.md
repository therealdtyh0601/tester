# Lumi Toolbox · Free · 阳宅卦生成器

🧭 3步生成阳宅卦象：**后天八卦 × 居住者 × 纳气口** → 64卦趋势。

## Free 版边界（已锁）
- ✅ 阳宅：卧室方位（内卦） + 门/阳台/窗（外卦）
- ✅ 输出：卦名 + 一句话趋势 + 低成本微调建议
- ❌ 不涉寻龙点穴（不看龙水砂穴）
- ❌ 不下吉凶断语 / 不做“改运”承诺

## 运行方式
### GitHub Pages
把整个目录上传到你的 repo 根目录或子目录，然后开启 Pages。

### 本地预览
因为使用 fetch 读取 JSON，建议用一个本地静态服务器，例如：
- VSCode Live Server
- `python -m http.server 8080`

然后打开：
`http://localhost:8080/`

## 数据
- `assets/data/gua64.json`：包含八卦资料 + 64卦基础解释（Free 版白话）
